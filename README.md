# 🚦 Sinaleira Interativa

Simulador de semáforo desenvolvido com **HTML, CSS e JavaScript puro**, permitindo a interação do usuário e a alternância entre diferentes modos de operação.

🔗 **Acesse o projeto online:**
[👉 Clique aqui para visualizar](COLOQUE_AQUI_O_LINK_DO_GITHUB_PAGES)

---

## ✨ Funcionalidades

* 🚥 Controle manual do semáforo (Abrir / Fechar)
* 🌙 Modo Noturno (luz amarela piscando)
* 🚨 Modo Emergência (luz vermelha fixa)
* 🔁 Cancelamento inteligente de temporizadores
* 🎯 Feedback visual para ações do usuário
* 📱 Layout responsivo

---

## 🧠 Lógica do Projeto

O sistema foi desenvolvido com foco em:

* Manipulação do DOM com JavaScript
* Controle de estado através de modos (`Normal`, `Noturno`, `Emergência`)
* Uso de `setTimeout` e `setInterval` para simular o comportamento real de um semáforo
* Gerenciamento de eventos e prevenção de conflitos entre ações do usuário

---

## 🛠️ Tecnologias Utilizadas

* JavaScript (Vanilla JS)
